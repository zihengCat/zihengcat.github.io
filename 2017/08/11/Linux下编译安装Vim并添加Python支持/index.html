Linux下编译安装Vim并添加Python支持

Posted by zihengCat on 2017-08-11

前言

Vim一直是子恒喵最钟爱的文本编辑器，为Vim添加Python支持可以使得Vim如虎添翼。
本文讲解在Linux系统下编译安装Vim并添加Python支持。

安装步骤

Step1 安装Python环境

Python有两个版本，Python2与Python3，我们可以选择安装任意版本，也可以把两个版本的Python都装上。

我们可以使用包管理器安装Python，也可以手动编译安装Python。手动编译安装具体步骤可以看这篇博文。
子恒喵的操作系统是CentOS7，Python2从yum仓库安装，Python3则手动编译安装。

Step2 获取源码

前往vim.org官网获取源代码，推荐直接克隆Vim在GitHub上的镜像仓库。

$ git clone https://github.com/vim/vim.git

Step3 解决依赖性

Vim所需的依赖很少，我们只需安装GCC，make等基础编译工具即可。至少子恒喵没有遇到啥依赖性的问题。如果遇到缺少依赖包(可能有个ncurses)，configure会提示我们，按照提示装好就行了。

Step4 添加配置选项

需要添加哪些功能，就把哪些功能选项打开。Ruby，Perl，Lua等等的动态语言支持功能都可以打开，前提当前系统已经安装好了相应的环境，这里我们只开启对Python支持。

./configure --with-features=huge \
            --enable-gui=no \
            --enable-pythoninterp=yes \
            --with-python-config-dir=/usr/lib64/python2.7/config \
            --enable-python3interp=yes \
            --with-python3-config-dir=/usr/local/packages/python-3.6.2/lib/python3.6/config-3.6m-x86_64-linux-gnu \
            --disable-selinux \
            --disable-sysmouse \
            --disable-nls \
            --prefix='/usr/local/packages/vim-80/' \
            CFLAGS='-O3  -Wall'

这是子恒喵的个人的编译配置，不添加 GUI 与多语言环境的支持
详细选项可以查看: ./configure --help

注意这几个选项:

选项 含义
--prefix 指定安装目录
--enable-pythoninterp=yes 打开Python2支持
--with-python-config-dir=<PATH> 指定Python3配置库路径
--enable-python3interp=yes 打开Python3支持
--with-python3-config-dir=<PATH> 指定Python3配置库路径

开启Python支持选项后，还需要指定Python配置库的路径。

Python的配置库路径一般在 <PATH>/lib/pythonx.x/config-xx/*下。其中<PATH>是Python的安装目录，具体路径，不同版本会略有不同。

Step5 编译安装

标准的编译安装流程。如果配置没有什么问题，可以顺利地编译安装。

$ make
$ sudo make install

如果自定义了安装目录，注意添加到系统<PATH>路径。

Step6 检验安装

$ vim --version
...

安装没问题的话，可以看到vim的版本信息。编译好的vim已经包含对Python的支持，并链接到Python的配置库。

:python import sys; print(sys.version)
:python3 import sys; print(sys.version)

进入vim，输入以上指令，如果可以顺利打印出Python版本号，说明vim的Python支持确实可用。
