Linux下编译安装Vim并添加Python支持

Posted by zihengCat on 2017-08-11

前言

本文讲解在Linux系统下编译安装Vim并添加Python支持。
Vim是子恒喵钟爱的文本编辑器，添加Python支持是为了使用大量优秀的Vim插件。

安装步骤

Step1 安装Python环境

Python有两个版本，Python2与Python3，我们可以选择任意版本安装，也可以把两个版本的Python都装上。

我们可以使用包管理器安装Python，也可以手动编译安装Python。手动编译安装具体步骤可以看这篇博文。
子恒喵的情况有些特殊(系统是CentOS7…)，Python2是从yum仓库安装的，Python3是编译安装的。

Step2 获取源码

前往Vim.org获取源码，推荐使用Vim在GitHub上的镜像。

Step3 解决依赖性

Vim的依赖很少，我们只需安装GCC，make等等的编译工具即可。至少子恒喵没有遇到依赖性的问题。如果遇到缺少依赖包(可能有个ncurses)，configure会提示我们，装好就行了。

Step4 编译配置选项

需要添加哪些功能，就把哪些功能选项打开。Ruby，Perl等功能都可以打开，前提是系统已经安装好了相应的环境，这里我们只开启对Python支持。

这是子恒喵的个人的编译配置，没有添加GUI与多语言环境的支持
详细选项查看: ./configure –help

注意这几个选项。

  • --prefix: 指定安装目录
  • --enable-pythoninterp=yes: 打开Python2支持
  • --with-python-config-dir=PATH: 指定Python3配置库路径
  • --enable-python3interp=yes: 打开Python3支持
  • --with-python3-config-dir=PATH: 指定Python3配置库路径

开启Python支持选项后，还要指定Python配置库的路径。Python的配置库路径一般在 /lib/pythonx.x/config-xx/ 下。PATH是Python的安装目录，具体路径，不同版本略有不同。

Step5 编译安装

标准的编译安装过程。如果没有什么问题，可以顺利地编译安装。

如果指定了安装目录，注意添加到系统PATH路径。

Step6 检验安装

安装没问题的话，可以看到vim的版本信息。vim已经包含Python功能，并有链接Python的配置库。

进入vim，输入以上指令，如果可以顺利打印出Python版本号，说明vim的Python功能确实可用。
