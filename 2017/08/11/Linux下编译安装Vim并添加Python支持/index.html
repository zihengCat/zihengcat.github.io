Linux下编译安装Vim并添加Python支持

Posted by zihengCat on 2017-08-11

前言

本文讲解在Linux系统下编译安装Vim并添加Python支持。
Vim是子恒喵钟爱的文本编辑器，添加Python支持是为了使用大量优秀的Vim插件。

安装步骤

Step1 安装Python3

子恒喵选择为Vim添加Python3的支持。我们可以使用包管理器安装Python3，也可以手动编译安装Python3。
具体步骤可以看这篇博文。
子恒喵是编译安装的。

Step2 编译安装Vim80

获取源码

我们可以前往Vim.org获取源码，也可以使用Vim在GitHub上的镜像。

解决依赖性

Vim的依赖很少，我们只需安装GCC，make等等的编译工具即可。
至少子恒喵没有遇到依赖性的问题。
如果遇到缺少依赖包，configure会提示我们，装好就行了。

应用配置选项

这是子恒喵的个人的编译配置啦
详细可以查看 ./configure –help

注意这几个选项。

  • --prefix: 指定安装目录
  • --enable-python3interp=yes: 打开Python3支持
  • --with-python3-config-dir=PATH: 指定Python3配置库的路径

开启Python3支持选项后，还要指定Python3配置库的路径。Python3的配置库路径一般在/lib/pythonx.x/config-xx/下。具体路径，不同版本可能略有不同。

编译安装

标准的编译安装过程。

如果指定的安装目录，注意添加一下系统PATH路径。

Step3 检验安装

安装没问题的话，可以看到vim的版本信息。可以看到，vim已经包含python3功能，并且提示有连接Python3的配置库。

新建一个pyv.vim文件，输入以上内容，在vim中:so %应用一下，再输入:pyv，没有问题的话，打印出了Python的版本号。
