Python学习笔记 - 包管理器pip使用

Posted by zihengCat on 2017-11-11

前言

pip是PyPA提供的一个非常好用的 Python 包管理工具。
本文记录一些pip的常用命令（pip3 适用）。

pip 常用命令

通用（General）

选项 含义
--help 打印帮助页面
--version 打印当前pip版本信息
--verbose 显示详细信息
--quiet 显示简略信息
--no-cache-dir 不使用缓存
--proxy <proxy> 使用代理

搜索（Search）

pip搜索命令，直接输入搜索信息，pip会在PyPI仓库中搜索相关的包。

$ pip search [options] <query>

安装（Install）

pip在线安装命令，全局安装可能需要 sudo 权限。

$ pip install [options] <package> ...

离线安装软件包（.whl是Python软件包打包格式）。

$ pip install <package.whl>

更新软件包，加入--upgrade选项即可。

选项 含义
-r 递归安装依赖包
--upgrade 更新已安装的包

卸载（Uninstall）

pip卸载命令，卸载已安装的包。

$ pip uninstall [options] <package> ...
选项 含义
--yes 直接确认卸载

列表（List）

pip列表命令，可以列出已安装包的相关信息。
注意一下，pip的检查更新功能放在list里。

$ pip list [options]
选项 含义
--outdated 列出有更新的包（检查更新）
--uptodate 列出已更新的包
--local 仅列出在虚拟环境中的包

显示（Show）

pip显示命令，查看已安装包的详细信息。

选项 含义
--files 显示该软件包中的所有文件

freeze

pip freeze命令，可以生成一份依赖包列表，列表格式可被pip install命令接受。常用来生成项目依赖。

更换pip源

pip使用PyPI仓库来为我们提供Python软件包。但是默认仓库地址在国内访问不顺畅，我们可以利用-i选项使用自定义镜像仓库。

PyPI默认仓库地址: https://pypi.python.org/pypi。

选项 含义
--index <url> 自定义 Python 包仓库地址

PyPI 国内镜像推荐

提供方 地址
阿里云 http://mirrors.aliyun.com/pypi/
豆瓣网 http://pypi.douban.com/
清华大学 https://pypi.tuna.tsinghua.edu.cn/
中科大 http://pypi.mirrors.ustc.edu.cn/

一些 PyPI 国内镜像

如何使用 PyPI 镜像

pip install -i <url/simple> some-packages

-i表示临时使用镜像地址
simple是 PyPI 软件包目录

修改pip.conf文件可将自定义镜像地址设为默认。

操作系统 文件位置
Linux ~/.pip/pip.conf
macOS $HOME/Library/Application Support/pip/pip.conf
Windows %APPDATA%\pip\pip.ini

操作系统不同，pip.conf文件位置不同
如果目录/配置文件不存在，手动创建即可

在pip.conf配置文件中全局指定PyPI镜像地址。

[global]
index-url = <url/simple>
