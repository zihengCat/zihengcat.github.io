Python包管理器Pip使用

Posted by zihengCat on 2017-11-11

前言

pip是PyPA提供的一个好用的 Python 包管理工具。
本文记录一些pip的常用命令（pip3 也适用）。

Pip 常用命令

通用(General)

选项 含义
--help 打印帮助页面
--version 打印当前Pip版本信息
--verbose 显示详细信息
--quiet 显示简略信息
--no-cache-dir 不使用缓存
--proxy <proxy> 使用代理

搜索(Search)

pip搜索命令，直接输入搜索信息，pip就会在PyPI仓库中搜索相关的包。

安装(Install)

pip安装命令，全局安装可能需要 sudo 权限。

离线安装软件包(.whl是Python软件包打包格式)。

选项 含义
--upgrade 更新已安装包

卸载(Uninstall)

pip卸载命令，卸载已安装的包。

选项 含义
--yes 直接确认卸载

列表(List)

pip列表命令，可以列出已安装包的相关信息。
注意一下，pip的检查更新功能放在list里。

选项 含义
--outdated 列出有更新的包（检查更新）
--uptodate 列出已更新的包
--local 仅列出虚拟环境中的包

显示(Show)

pip显示命令，查询已安装包的详细信息。

选项 含义
--files 显示该软件包中的所有文件

更换Pip源

pip默认的PyPI仓库是: https://pypi.python.org/pypi。
默认有时在国内访问不顺畅，我们可以利用-i选项自定义镜像仓库。

选项 含义
--index <url> 自定义 Python 包仓库

一些 PyPI 国内镜像

提供方 地址
阿里云 http://mirrors.aliyun.com/pypi/
豆瓣 http://pypi.douban.com/
清华 https://pypi.tuna.tsinghua.edu.cn/
中科大 http://pypi.mirrors.ustc.edu.cn/
