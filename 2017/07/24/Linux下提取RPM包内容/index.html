Linux下提取RPM包内容

Posted by zihengCat on 2017-07-24

前言

对于红帽（RedHat）系的 Linux 系统而言，大多使用rpm作为系统包管理工具（RPM Package Manager）。
我们可以使用命令rpm -qlp来查看一枚rpm包所包含的文件。那么，如何在不安装rpm包的情况下提取包中的文件呢？

提取rpm包内容

为了提取rpm包中的文件，我们需要用到两款工具：rpm2cpio与cpio。
使用如下命令，可将rpm包内容（文件）提取出来：

$ rpm2cpio <package_name.rpm> | cpio -idmv

代码清单：提取rpm包中的文件

接下来，对命令含义做简要的描述。

  • rpm2cpio

rpm包默认使用cpio格式进行归档打包，此工具就是将rpm包转换为cpio归档，命令默认直接输出到标准输出（stdout）。

  • cpio
    cpio工具主要用来建立或者还原备份归档，与tar工具类似。
    • -i
      extract，导出文件。
    • -d
      make-directories，建立新目录存放导出文件。
    • -m
      保持文件的更新时间。
    • -v
      verbose，显示详情。
