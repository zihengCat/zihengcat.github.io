macOS 设置终端 socks5 代理连接

Posted by zihengCat on 2018-02-27

前言

终端是每位程序员的好伙伴，但是由于某些众所周知的原因，国内恶劣的网络环境让程序员们只能看着终端上提示信息[连接断开，下载失败…]欲哭无泪。
本文介绍macOS下设置终端走socks5代理连接的方法（也适用于UNIX/Linux）。

终端设置

在终端下，设置环境变量ALL_PROXY为本地socks5代理连接。

$ export ALL_PROXY=socks5://127.0.0.1:<port>

<port>: 本地代理连接端口号。

我们可以通过观察IP来判断代理连接是否设置成功。

$ curl -i http://ip.cn

脚本化设置

我们可以将命令写入到~/.bash_profile中，快捷设置。

function setProxy() {
    export ALL_PROXY="socks5://127.0.0.1:${1}"
}
function unsetProxy() {
    unset ALL_PROXY
}
function testProxy() {
    curl -i 'http://ip.cn'
}

使用shell函数，alias别名都可以实现。
这里演示shell函数实现。

$ setProxy <port>   # 启动终端代理连接
...
$ unsetProxy        # 关闭终端代理连接

注意事项

设置环境变量ALL_PROXY后，终端下只要支持此环境变量的网络应用程序大多都会走这个代理（如：　Homebrew）。还有，socks5代理会与国内镜像源发生冲突，也是因为某些众所周知的原因…
