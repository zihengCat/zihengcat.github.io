Vim学习笔记-常用查找替换命令

Posted by zihengCat on 2018-01-03

前言

本文记录Vim下一些常用的查找与替换命令。

查找(Search)命令

查找操作通过/或?开启，输入查找字符串后按<Enter>回车键进入查找模式，按<Esc>退出键退出查找模式。

Command Description
/word 正向(上)查找目标字符串
?word 反向(下)查找目标字符串

如果查找到多个匹配项，可通过n(前一项)与N(后一项)来回跳转。
查找后，匹配字符串会高亮反显出来，输入:noh或再搜索一枚文件中不存在的字符串就可以取消高亮啦。
